# Data-Science-Salary-Dashboard-
An interactive Excel dashboard that visualizes Data Science job Salaries by job title, country, and employment type, with a global salary distribution map and median
Using a Data Validation on choosing the job title & Job Country & Employment Type. 
We used a KPI cards to filter  Which Job_title has a median Salary based based on Job Country & Employment type also based on  in Which Top Jobs platforms for Data Science roles
and also Top Job Counts based on filters 
We Used a bar chart to visualize the job title & Employment ID Which aldo we used a Map Charts for visualizing the Countries.
